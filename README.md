# FitnessProgram

A simple Java program to manage personalized fitness workouts and track user progress and calories.

# Features

- Store user information (name, age, height, weight)
- Create multiple workouts with sets and reps
- Organize workouts into a program
- Display user info and workout program in the console

# Project Structure

- `Main.java` → Entry point to run the program  
- `User.java` → Class to store user information  
- `Workout.java` → Class to store individual workouts  
- `Program.java` → Class to manage a list of workouts  

# How to Run

1. Open the project in **BlueJ**.  
2. Compile all classes.  
3. Right-click `Main` → Select `void main(String[] args)` → Click OK.  
4. View output in the console.

# Future Improvements
